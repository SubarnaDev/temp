🚀 Welcome to BasicPay+

BasicPay+ is a blazing-fast Stellar wallet built for gig workers, small merchants, and remote communities. It features smart payments, encrypted wallet storage, recurring transfers, and even microloans — all in one lightweight web app.

  • ✅ Generate & encrypt your wallet
  • 💸 Send payments or schedule recurring ones
  • 📎 Request money with QR codes
  • 🤖 Ask BasicBot for help at any time
  • 📊 View balance charts & transaction history
  • 🌍 Offline support for limited-connectivity zones

🔁 Try it live — no signup needed. Fund your wallet using Stellar Friendbot and start transacting!
